### Q8. 다음 코드의 문제점은?
```java
int a = Integer.MAX_VALUE;
int b = 10;
		
do {
	a++;
}while(b < 5);
```
<details>
<summary>정답</summary>

`do-while`문은 조건에 관계없이 한번은 실행되기 때문에 a는 `-2147483648`이라는 값이 나오게 된다.
</details>

### Q9. 다음 중 옳지 않는 것은?

3.   배열의 모든 요소는 같은 형태(type)이어야 한다.

</details>
